## `,` comma operator

- `,` 는 피연산자임
- `a , b` 일 때 a 와 b는 피연산자
- `,` 는 각각의 피연산자를 평가하고 가장 오른쪽에 있는 피연산자를 리턴함

```js
let a = 1,
  b = 2; // return b => b는 2로 평가 => 2 출력

a, b; // a 피연사자 평가, b 피연산자 평가, 가장 오른쪽의 b retrun => b는 2로 출력
```
